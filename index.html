Presentation

flagSVG.js is a JavaScript library to insert flags into your pages.

SVG images of the flags are retrieved from WikiMedia and are in the public domain.

Normal Rounded

How to use it?

Download the file containing all the flags, or create your own file with the flags needed.

Download all flags

Include this code in your page:

<svg class="flag" data-flag="us" width="400" height="200"></svg>

Edit the data-flag attribute to specify the flag code.

<script src="https://code.jquery.com/jquery.js"></script>
<script src="flagSVG.js"></script>
<script>
  $(document).ready(function() {
    flag(".flag");
  });
</script>

This code loads the jQuery library and the flagSVG.js file.

Once the page is ready, the flag() function is called. Its parameters allow you to choose what will be the class of the svg to load the flags.
